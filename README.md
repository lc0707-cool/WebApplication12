# WebApplication12

# 🛡️ 企業 項目風險評估 平台 

## 🎓 專案簡介（自主學習成果）

本專案為自主學習實作成果，透過 **ASP.NET Core MVC** 架構，結合 **資料庫驅動設計（Data-Driven Design）** 與基本的 **前後端非同步互動（Ajax）技術**，建立一套資安風險評估的管理系統。

系統模擬企業內部資訊資產在不同角色下的管理流程，涵蓋：

> 項目登錄 → 風險評估 → 狀態更新 → 風險定義維護 → 資料回收管理
>
> ### 1. 📊 項目列表（Index）-> 資安風險評估作業平台
- 提供項目清單檢視與管理
- 支援風險標籤（多選）與風險等級設定（高 / 中 / 低）
- 支援處置狀態更新（未處理 / 處理中 / 已緩解）
- 使用 Ajax 即時更新資料庫，無需重新整理頁面
- 支援基本條件查詢與快速定位

---

### 2. 🔍 即時盤點作業介面（單筆查詢）
- 透過項目 ID 查詢單筆項目資料
- 自動載入該項目風險狀態與已勾選項目
- 使用 JSON 回傳資料，提高前後端互動效率
- 適用於巡檢或資產盤點情境

---

### 3. 📥 新增項目
- 提供基礎項目建檔功能
- 自動產生初始風險代碼（預設 X0）
- 預設風險等級與處置狀態
- 使用資料庫 Identity 自動編號

---

### 4. ⚙️ 風險定義管理
- 可新增資安風險定義項目
- 系統自動產生風險代碼（如 X1、X2…）
- 風險定義即時同步至所有評估介面
- 提供風險清單檢視功能，確保標準一致性

---

### 5. ♻️ 資料回收桶（Soft Delete 機制）
- 採用軟刪除機制（IsDelete）
- 可還原已刪除資產
- 支援永久刪除功能

---

## 🛠️ 技術架構

- **後端框架**：ASP.NET Core MVC (.NET 6)
- **資料庫**：SQL Server LocalDB
- **資料存取**：ADO.NET（Microsoft.Data.SqlClient）
- **前端技術**：HTML / CSS / Bootstrap 5 / jQuery
- **非同步技術**：Ajax（JSON API）

---

## 💡 系統特色

- 資料庫驅動（Data-Driven）設計架構
- 前後端分離式 API（JSON 回傳）
- 即時更新資料（無刷新操作）
- 軟刪除與資料回復機制
- 模擬企業資安風險管理流程
